What is this website?

I'm building a friendly way to experience the Macintosh gaming experience of the 1990s and early 2000s "Classic" era. This site includes a collection of {{ count }} downloadable Macintosh game demos, {{ infinitemac }} of which can be played directly in a web browser courtesy of Infinite Mac.

You'll also find {{ sources }} CD-ROM disc images, mostly coverdiscs from Macintosh magazines of the era. All the CDs can be explored in an Infinite Mac emulator. The CDs on this site are a combination of discs I've collected and contributed to Internet Archive, as well as disc images others have contributed to the Internet Archive, Macintosh Garden, and Macintosh Repository.

Many of these games aren't abandonware! Remasters and modern adaptations are available for a lot of them, often offering significant quality-of-life improvements. Whenever possible, this site includes links to buy and play the full games, as well as to modern open-source builds from Mac Source Ports. Even the games that are now abandonware often used serial keys, copy protection, or DRM that's a nuisance to work around. Demos were DRM-free - just click and play!

How do I play one of these demos?

In most cases, just click " Play". If there's no play button for a game, it can be downloaded and played using a Classic-era Macintosh computer or an emulator. The guides at Emaculation.com are an excellent way to get started with emulators. This site offers an http-only version accessible to vintage web browsers. Use Stuffit Expander to decode and uncompress the downloads. Retro Mac OS X fans can enjoy many of these games in "Classic" mode on systems running Mac OS X versions up to and including Mac OS X 10.4 Tiger.

How do I get in touch with you?

@kabootie@oldbytes.space created and maintains this site. If you're a rightsholder who would like the demo for your game removed, you can request that by emailing i_hate_game_demos@classicmacdemos.com.
