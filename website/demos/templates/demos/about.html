What is this website?

I'm building a user-friendly way to experience Macintosh videogames from the 1990s "Classic" era. This site collects {{ count }} downloadable game demos, {{ infinitemac.count }} of which can be played directly in a web browser courtesy of Infinite Mac.

It also collects {{ sources}} Macintosh CD-ROM images from the Classic Macintosh era, all of which can be launched from the site into an emulated Macintosh at InfiniteMac for easy exploration. The CD-ROM images on this site are a combination of discs I've collected and contributed to Internet Archive, as well as disc images others have contributed to the Internet Archive, Macintosh Garden, and Macintosh Repository.

Many of these games aren't abandonware! Remasters and modern adaptations are available for a lot of them, often offering significant quality-of-life improvements. Whenever possible, this site includes links to buy and play the full games, and to modern open-source builds from Mac Source Ports. Even the games that are now abandonware often used serial keys, copy protection, or DRM that's a nuisance to work around. Demos were DRM-free - just click and play!

How do I play one of these demos?

In most cases, just click " Play at Infinite Mac". If that's not available for a game, play using a Classic-era Macintosh computer or an emulator. The guides at Emaculation.com are an excellent way to get started with emulators. This site offers an http-only version accessible to vintage web browsers. Use Stuffit Expander to decode and uncompress the downloads. Retro Mac OS X fans can enjoy many of these games in "classic" mode on systems through Mac OS X 10.4 Tiger.

How can I support this project?

I don't ask for financial support for this website. Instead, please join me in supporting the people, projects, and communities that make it possible: Internet Archive, Infinite Mac, Macintosh Garden, Macintosh Repository, Wikimedia Foundation, Django, The Trevor Project.

Anyone is welcome to contribute to the content on this website. See below for a few areas where help would be appreciated, and contact me if you'd like to collaborate on any of these.

Make an Infinite Mac disk image for these {{custom_image.count}} games

Some games are playable at Infinite Mac, but the experience is clunkier than it could be. The game mounts in a disk image on the Mac desktop that you need to launch it from. A better experience is a custom bootable image with a minimal System folder and set of extensions, and an alias to the game in Startup Items so it launches automatically at startup, like this. These {{custom_image.count}} games were added before Infinite Mac supported custom boot images, so they don't take advantage of the better experience that makes possible. Let's fix that! The best way to do this is to create disk images in the Saved HD drive, try booting from it, then send me the exported drive once you get it working.
